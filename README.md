## 📖 简介

**辩论区整活大师** 是一个基于 Google Gemini 3 Pro 模型构建的趣味 AI 聊天应用。

在这个充满戾气的互联网时代，我们不需要另一个一本正经的说教机器。我们需要的是一个 **1992 年出生、月入三千、看透生活本质的“整活老哥”**。

不管你是想回怼杠精、化解尴尬，还是单纯想找个“嘴替”吐槽生活，它都能用一套独有的“歪理邪说”和自嘲式幽默，把严肃的争吵变成让人会心一笑的段子。

## ✨ 功能亮点

- **🎭 鲜明的人设调教**：精心设计的 Prompt，不仅是回答问题，更是进行一场“行为艺术”。它擅长用自嘲破冰，用幽默收场。
- **🧠 强力模型驱动**：底层接入 **Gemini 3.0 Pro Preview**，推理能力更强，懂梗、懂语境，反应够快。
- **💬 丝滑的交互体验**：
  - 极简 UI 设计，专注对话。
  - 自动滚动与防遮挡优化，打字体验极佳。
  - 支持“一键重置”，随时开启新话题。
- **🎨 现代化技术栈**：使用 React 19 + Tailwind CSS 构建，轻量且高性能。

## 🛠️ 安装与使用

本项目是一个纯前端 React 项目，配置简单，上手即用。

### 前置要求
你需要一个 Google Gemini 的 API Key。

### 1. 克隆项目
```bash
git clone https://github.com/your-username/debate-peacemaker.git
cd debate-peacemaker
```

### 2. 安装依赖
```bash
npm install
# 或者
yarn install
```

### 3. 配置环境变量
在项目根目录下创建一个 `.env` 文件（或修改现有的），填入你的 API Key：

```env
API_KEY=your_google_gemini_api_key_here
```

### 4. 启动项目
```bash
npm start
```
打开浏览器访问 `http://localhost:3000` 即可开始整活！
